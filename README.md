​📌 About The Project
​DVCI AI Copilot is a proactive RegTech platform engineered to automate digital financial investigations and ensure real-time compliance with the Saudi Central Bank (SAMA) regulations. By leveraging a hybrid architecture that merges the deterministic precision of C++ risk-scoring math with the contextual reasoning depth of Generative AI, DVCI slashes administrative case-building times from days down to just 10 seconds—delivering a massive 90% boost in operational efficiency.
​🤖 Multi-Agent Architecture
​The system functions as a collaborative ecosystem of parallelized, task-specific AI agents:
• ​Financial Agent: Continuously analyzes transactions, flags behavioral baseline anomalies, and computes dynamic Risk Scores.
• ​Compliance Agent: Automatically maps all flagged activities against standard localized legal and AML frameworks.
• ​Investigation Agent: Correlates multi-party metadata instantly to generate an immutable, contextualized case file.
• ​Interrogative Copilot: An interactive, conversational interface that allows human analysts to query the core logic for transparent, evidence-based cross-examination.
​🛠️ Tech Stack
• ​Core Systems: Python & C++ (Optimized for real-time transaction ingestion and low-latency risk-scoring engines).
• ​Intelligence Layer: Multi-Agent AI frameworks & Machine Learning predictive modeling.
• ​Inference Engine: Advanced LLMs for secure diagnostic logic and forensic reporting.
• ​Frontend Operations: UI/UX built via Google interactive design frameworks.
• ​Data Integrity:SHA-256 cryptographic hashing to ensure chain of custody and secure forensic report tampering protection.
​🎯 Governance & Privacy
• ​Human-in-the-Loop (HITL): The system operates strictly as an advanced recommendation advisor, ensuring final enforcement remains bound to human analysts.
• ​Data Privacy: To adhere strictly to institutional banking secrecy, the platform utilizes high-fidelity Synthetic Data Scenarios to test compliance workflows safely.
